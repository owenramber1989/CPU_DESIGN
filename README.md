# CPU_DESIGN

## 任务分配

### :musical_keyboard: 音频处理(zjm)

- 音频下载与转换
- 谱面设计
- 音频信号发生器
- 频率信号生成逻辑
- 蜂鸣器(实现声音的播放)

### :computer: 中央处理器(ly)

- 撰写汇编代码

### :tv: 用户界面(cly)

- 界面图像发生器
- 界面显示器（提示信息：当前得分、miss、hit 等等）
- 成果汇报 ppt 制作
- 演示视频的录制

### :musical_score: 节拍处理(nzh)

- 节拍图像信号生成逻辑
- 节拍显示器（实现谱面的显示）
- 建立代码仓库并进行管理
- 播放器设计 击打歌曲的同时播放 mv 视频

## features

- 多 key 谱面设计与添加(可保留单 key，把多 key 设置为第二个难度，或者直接添加)
